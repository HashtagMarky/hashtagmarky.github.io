# #M Productions — Cross-Posting Strategy

**Version 1.0 · May 2026**

-----

Cross-posting is the practice of publishing the same piece of content to multiple platforms with minimal re-editing. The goal is to maximise reach from a single piece of content without building a separate production workflow for each platform.

**Current platforms:** Instagram (primary) → TikTok (secondary)

Content flows in one direction: produce for Instagram first, then adapt for TikTok. Never produce TikTok-first.

-----

## When to Cross-Post

Not everything warrants cross-posting. Use this filter:

| Cross-post? | Criteria |
|---|---|
| Yes | Higher-performing Instagram posts (above-average saves, shares, or reach) |
| Yes | Carousels — they translate directly to TikTok slideshows with no re-editing |
| Yes | Reels that are visually self-contained (understandable without prior context) |
| No | Stories — not indexable on either platform, no point duplicating |
| No | Posts with heavy Instagram-specific text (e.g. "link in bio" as the main CTA) |
| No | Low-performing posts — cross-posting a dud just spreads underperformance |

-----

## Workflow

### Step 0 — Export without watermark

Before posting anywhere, export the final file from CapCut or your editing app. Do not download from Instagram to repost — the watermark suppresses reach on TikTok.

If you've already posted to Instagram and need to retrieve the file, use SaveIG (saveing.com) to download watermark-free.

### Step 1 — Post to Instagram

Instagram is always first. Use the full caption structure from [instagram-strategy.md](instagram-strategy.md):
- Keyword-rich first line
- 2–4 line body
- CTA pointing to the site
- 5–10 hashtags at the end

### Step 2 — Wait 24–48 hours

Post to TikTok the day after Instagram. Simultaneous posting provides no benefit and removes the option to test Instagram performance before committing to cross-post.

### Step 3 — Adapt for TikTok

Two things change between platforms:

| Element | Instagram | TikTok |
|---|---|---|
| Caption | Full structure (hook + body + CTA + hashtags) | 1–2 sentence hook only |
| Hashtags | 5–10, broad/mid/niche mix | 3–5, all relevant, include platform tag (`#BookTok` etc.) |

Nothing else changes. The media file is identical.

-----

## Platform Differences to Keep in Mind

| | Instagram | TikTok |
|---|---|---|
| Primary format | Carousels for saves, Reels for reach | Slideshows and video equal weight |
| Audience overlap | Minimal — treat as separate audiences |  |
| Hashtag function | Niche classification + reach | Algorithm classification — use consistently |
| Caption weight | First line indexed for search | Low — visual hook carries more weight |
| Optimal post gap | — | 24–48 hrs after Instagram |
| Watermark policy | N/A | Suppresses reach if IG watermark present |

## Content Mapping by Pillar

### Books

| Instagram | TikTok |
|---|---|
| 5-slide carousel (cover → summary → take → who's it for → verdict) | Same slides, uploaded as photo slideshow |
| Caption: title + author + one-word take + CTA | Caption: title + author + one-line opinion |
| Tags: `#bookstagram` `#scifibooks` `#fantasybooks` `#bookreview` | Tags: `#BookTok` `#SciFiBookTok` `#FantasyBookTok` |

### ROM Hacking

| Instagram | TikTok |
|---|---|
| Reel (30–60s walkthrough) | Same video, trimmed if over 60s |
| Caption: episode + summary + "devlog at link in bio" | Caption: one sentence describing what's shown |
| Tags: `#pokemonromhack` `#pokeemerald` `#romhacking` | Tags: `#PokemonRomHack` `#RomHacking` `#PokemonEmerald` |

### Tech/Builds

| Instagram | TikTok |
|---|---|
| Carousel (before/during/after) or Reel | Same format |
| Tags: `#buildlog` `#raspberrypi` `#diygameboy` | Tags: `#BuildLog` `#RaspberryPi` `#DIYGameboy` |

-----

## Cross-Posting Checklist

- [ ] Post performed above-average on Instagram (or is a carousel worth testing)
- [ ] File exported from editor without watermark — not downloaded from Instagram
- [ ] Waited 24–48 hours after Instagram post
- [ ] Caption trimmed to 1–2 sentences for TikTok
- [ ] Hashtags swapped to TikTok set (3–5 tags, platform tag included)
- [ ] Text overlays on every slide are readable without sound
